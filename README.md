Instructions on running the 401k manager:

1. Clone repository from Github
2. Files needed
    user_mock_data.json(json): contains user mock data
    server.py(python) : server to display and retrieve data
    frontpage.html(html and css) :  design of web page
    human_interest.js(JavaScript): core functionality, contains button logic, calculation logic, etc. 

3.Open your terminal and download flask:
- Either set up a python environment 
Use the following commands 
    python3 -m venv venv
    source venv/bin/activate
    pip install flask
    pip list
- or download without an environment 
    pip install flask

4. Type in 
    python3 server.py into your terminal to start the server
5. In your browser type in:
    http://localhost:5560/
    to connect 
    
6. Welcome to the app! 

7. to deactive your virtual environment: 
    deactivate
